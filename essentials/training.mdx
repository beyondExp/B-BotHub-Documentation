# Text Tutorial

## Train your Expert

To train your expert you have to go to the training page.
Here you can choose the expert you want to train.
<img
  style={{ borderRadius: '0.5rem' }}
  src="/images/training-first.png"
/>

### Choose the model

You can choose the model you want to train.
Your Expert has a Basic model, if you already had a training Session you will be able to train on top of the trained model.
Click on your desired model and on the "Select Model" to get to the next step.

<img
  style={{ borderRadius: '0.5rem' }}
  src="/images/training-second.png"
/>

### Choose the base sentence

You can choose the base sentence you want to train. Theres is the option to train on the personality or on the abilities.
Choose your desired base sentence and click on it to get to the next step.

<img
  style={{ borderRadius: '0.5rem' }}
  src="/images/training-third.png"
/>

### Enrich your Training Data


<img
  style={{ borderRadius: '0.5rem' }}
  src="/images/training-fourth.png"
/>

There are multiple options to enrich your training data.
These options are a help to get to training data faster.



#### F2TD
Fact 2 Training Data is our own tool to get training data faster.
you can copy any text into the text field and click on the "Generate" Button to generate new data.
The model will generate training data for you.

<img
  style={{ borderRadius: '0.5rem' }}
  src="/images/training-fifth.png"
/>

Our Generator will create Conversation data for you to choose from.
You can choose the data you want to train on by clicking on the "Select" Button to add it to your training data.
Or you can click on the "Save to Favorites" Button to generate new data.

<img
  style={{ borderRadius: '0.5rem' }}
  src="/images/training-sixth.png"
/>

#### Favorites

You can see all of your saved training data in the favorites tab.
You can choose the data you want to train on by clicking on the "Select" Button to add it to your training data.


<img
  style={{ borderRadius: '0.5rem' }}
  src="/images/training-seventh.png"
/>

#### Sessions

You can see all of your training sessions in the sessions tab. You can see the name of the session and the date of the session as well as the description you gave to the session.
by clicking on it you will see all the Situations that were trained in this session.
you can select all the Situations you want to train on by clicking on the "Select" Buttons to add it to your training data.

<img
  style={{ borderRadius: '0.5rem' }}
  src="/images/training-eight.png"
/>

#### Import

You can import training data from a file. You can choose the file you want to import and click on the "Import" Button to import the data.
At the moment you can only import the Backups of ChatGPT.

<img
  style={{ borderRadius: '0.5rem' }}
  src="/images/training-ninth.png"
/>

### Edit the Situations

Click on Proceed to get to the next step where we will edit the Situations.

#### Name and Description

You can give the Training Session a name and a description.
This will help you to find the training session later on, and it will give you a better overview of what you trained on.

<img
  style={{ borderRadius: '0.5rem' }}
  src="/images/training-tenth.png"
/>

#### Add new Situations

You can add new Situations to your training data.
There are 2 Buttons for that, one on the top and one on the bottom.
By clicking it you will add a new Situation to your training data.

<img
  style={{ borderRadius: '0.5rem' }}
  src="/images/training-eleventh.png"
/>

#### Context Menu of a Situation

By clicking on the three dots of a Situation you can see the following functions:

- Duplicate: Duplicate the Situation
- Delete: Delete the Situation

<img
  style={{ borderRadius: '0.5rem' }}
  src="/images/training-twelfth.png"
/>

#### The Message Card

In each Situation there are multiple Message Cards. Each Message Card is a message that was sent to the expert or from the Expert.

*The User Message*
<img
  style={{ borderRadius: '0.5rem' }}
  src="/images/training-thirteen.png"
/>

Each message has a context menu. By clicking on the three dots of a message you can see the following functions:
- Remove: Remove the message from the Situation
- Toggle Role: Change the role of the message from User to Expert or from Expert to User
- Copy Content: Copy the content of the message
- Move Up: Move the message up in the Situation
- Move Down: Move the message down in the Situation

<img
  style={{ borderRadius: '0.5rem' }}
  src="/images/training-fifteen.png"
/>

*The Expert Message*

<img
  style={{ borderRadius: '0.5rem' }}
  src="/images/training-fourteen.png"
/>

The same context menu is available for the Expert Message. with one additional function:

- Add Action: Add an action to the message. This will be used to train the expert to do something after this message.

<img
  style={{ borderRadius: '0.5rem' }}
  src="/images/training-sixteen.png"
/>

if you click on the "Add Action" Button you will see the following window:

<img
  style={{ borderRadius: '0.5rem' }}
  src="/images/training-seventeen.png"
/>

An added action will look like this:

<img
  style={{ borderRadius: '0.5rem' }}
  src="/images/training-eighteen.png"
/>

Every tool will have an input field where you can determine the input of the action.
After you run the tool by clicking on the "Run" Button the output will be shown in the output field.

<img
  style={{ borderRadius: '0.5rem' }}
  src="/images/training-nineteen.png"
/>

### Train the Expert

After you have added all the Situations you want to train on you can click on the "Proceed" Button to get to the next step.

in there you can see a button to start the training.

<img
  style={{ borderRadius: '0.5rem' }}
  src="/images/training-twenty.png"
/>

After you have started the training you can see the progress of the training in the Training Button itself.

<img
  style={{ borderRadius: '0.5rem' }}
  src="/images/training-twenty.png"
/>
